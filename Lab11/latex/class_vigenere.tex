\doxysubsection{Detailed Description}
Class for encrypting and decrypting messages using the Vigenère cipher. 

The Vigenère cipher is a method of encrypting alphabetic text by using a simple form of polyalphabetic substitution. A key is used for the encryption and decryption of messages.
